==== Editor Fly Camera ====
Allows the game camera to be controlled like the editor Scene View camera (using right-click-drag + WASD).
Just slap it on your camera and it will start working.
Very useful for game jams or prototyping.

Controls: 
- WASD to move
- Q / E are up / down
- Hold Shift to speed up
- Pan with the middle mouse button
- Rotate with the right mouse button
- Scroll wheel - zoom
